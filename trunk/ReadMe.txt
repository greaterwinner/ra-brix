                Ra-Brix a Modular based Web Application Framework


********************************************************************************************

Ra-Brix is a modularized enabling web application framework. What that means is
in effect that Ra-Brix makes it possible to create extremely modularized web applications.

The website for Ra-Brix is; http://ra-brix.org
